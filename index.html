data.arxiv.io

This site is a repository for a dataset containing public metadata about abstracts listed on arxiv.org coerced into a form suitable for projects in natural language processing.

Data Format & Access

The data are saved in gzipped tab-separated ASCII files. Each abstract is a single line with the columns: ArXiv ID, space-separated categories, tokenized title, and tokenized abstract. The title and abstract have been tokenized using the NLTK implementation of the Penn Treebank tokenizer.

The files are available in the /data directory with URLs of the form /data/{category}/{year}-{month}-{day}.txt.gz. For example, all of the abstracts listed in the astro-ph category on August 6, 2007 are in the file /data/astro-ph/2007-08-06.txt.gz.
